# Miami GP 2025 — FastF1 Race Analytics

F1 race analysis notebook using FastF1 library with 9+ visualizations.

## Setup

```bash
pip install -r requirements.txt
jupyter notebook miami_gp_2025_analysis.ipynb
```

## Features
- Race classification & points
- Lap time evolution
- Tyre strategy
- Telemetry & speed traces
- Position changes
- Pit stop analysis
- Weather data

## Data Source
FastF1 library (live F1 telemetry)
